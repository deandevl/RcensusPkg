# RcensusPkg
Contains R script for obtaining data from the US Census API.

To install *RcensusPkg* enter the following command lines from an R console:

```R
install.packages("devtools")
```

```R
devtools::install_github("deandevl/RcensusPkg")
```

The repository contains an html document under the `demos/An_Introduction` folder that introduces the package along with numerous demonstrations of the individual functions.
